If you want to run directly, you can type: python paint.py
Make sure that you've installed Tkinter module for Python.

This Paint Application was written by Python 2.7

Convert python to exe, type: python setup.py py2exe
Caution: copy the 'pic' folder into 'dist' folder then the file paint.exe can load it.
Ref: http://www.py2exe.org/index.cgi/Tutorial
